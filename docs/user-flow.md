# User Flow

```text
หน้าแรก
  → เริ่มวิเคราะห์
  → เลือกเป้าหมาย + ประเภทอาคาร
  → เพิ่มบิลค่าไฟ ───────────────┐
  → สร้าง/นำเข้ารูปแบบการใช้ไฟ ├→ ตรวจสอบความพร้อม
                                  ├→ ค่าไฟและ TOU
                                  ├→ Solar
                                  └→ สรุปและรายงาน
```

Battery, EV และการวิเคราะห์เพิ่มเติมเป็นเส้นทางรองหลังผู้ใช้เข้าถึงผลหลักแล้ว ไม่อยู่ใน checklist หลักของเจ้าของบ้าน

## Guardrails

- ทุกขั้นใน checklist เป็นลิงก์ย้อนกลับไปแก้ข้อมูลได้
- หากยังไม่มีบิล/Load Profile หน้าวิเคราะห์ใช้ empty state และพาไปยังแหล่งข้อมูลที่เหมาะสม
- URL เดิมยังคงใช้งานได้; navigation เปลี่ยน label และโครงสร้างเท่านั้น
