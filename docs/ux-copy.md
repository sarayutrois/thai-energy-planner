| บริบท | ข้อความที่ใช้ |
| --- | --- |
| เริ่มต้น | “เลือกเป้าหมายและข้อมูลที่มี ระบบจะพาไปทีละขั้น” |
| ไม่มี Load Profile | “ยังไม่มีรูปแบบการใช้ไฟ — สร้างจากเครื่องใช้ไฟฟ้า หรือนำเข้าไฟล์จาก Smart Meter” |
| ข้อมูลบิลไม่ครบ | “เพิ่มบิลอย่างน้อย 1 เดือน โดยกรอกจำนวนหน่วยและค่าไฟรวมจากบิล” |
| พร้อมวิเคราะห์ | “ข้อมูลพร้อมสำหรับการประเมินเบื้องต้น — เพิ่มบิลย้อนหลังเพื่อเพิ่มความน่าเชื่อถือ” |
| ผลลัพธ์ | เริ่มด้วยคำแนะนำ, ผลประหยัดโดยประมาณ, ข้อจำกัด และขั้นตอนถัดไป ก่อนรายละเอียดเชิงเทคนิค |
| ความผิดพลาด | บอกสิ่งที่แก้ได้และตำแหน่งข้อมูลที่ต้องใช้; ไม่แสดง error ทางเทคนิคแก่ผู้ใช้ทั่วไป |
